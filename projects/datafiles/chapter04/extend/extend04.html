Color Converter Used: Google Color Picker

Hex uses three hexadecimal numbers to assign an amount of red, green, and blue to a color. RGB is the same, except it uses decimal numbers and can use a fourth value for opacity. HSL is newer, and functions much differently. It uses 3 numbers like the others, however they represent hue, saturation, and lightness. HSL can also use a fourth value for opacity. HSL was designed to be read more easily by humans, but I think RGB is easier to read.

Preferred color code format: RGB

HWB uses three numbers to represent hue, whiteness, and blackness.

CMYK uses four numbers to represent cyan, magenta, yellow, and key (black). It is based on the subtractive color model used in color printing.
